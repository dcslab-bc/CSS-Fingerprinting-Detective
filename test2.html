Welcome

This is a small, normal-looking page made to sanity-check your CSS fingerprinting detector. It includes a dark-mode media query, a touch-capability media query, and a simple width-based query. Each condition has a tiny background image URL that echoes the detected value(s).

Get Started

Tips

  • Toggle OS dark mode to trigger the dark rule.
  • Simulate a touch device (no hover, coarse pointer) to trigger the capability rule.
  • Resize the window beyond 900px to trigger the layout rule.
